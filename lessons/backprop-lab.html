Backprop Lab

A whole tiny network on one screen — two inputs, two hidden units, one output, one loss. Watch numbers flow right on the forward pass, then gradients flow left on the backward pass, exactly as the chain rule prescribes. Then press Train and watch every weight crawl downhill.

Backprop feels like magic until you see it as bookkeeping. Every node knows its own value (forward, in white) and how much the loss would change if it wiggled (backward, in red). Press Gradient step and watch each weight move against its gradient — and the loss shrink.

Read one red number. Click nothing — just look at the gradient under a weight. If it's positive, nudging that weight up would raise the loss, so gradient descent pushes it down (and vice-versa). Every weight update is just: move a little in the direction that lowers the loss. Backprop's only job is to compute all those red numbers efficiently, in one backward sweep.

Try these

1. Do one step by eye. Note the loss. Press Gradient step ×1 once. The loss drops a little and every weight changed by −(learning rate)×(its red gradient). That single move, repeated, is all of training.

2. Break it with the learning rate. Push the learning rate to 2 and hold Train. The loss may overshoot and bounce instead of settling — the same divergence you saw in the gradient-descent lab, now inside a real network.

3. Make it impossible, then possible. Set the target and inputs so the network struggles, then Train — watch it find weights that work anyway. Two hidden units are enough to bend the output wherever the loss needs it.

What this lab proves
  • Forward makes predictions, backward assigns blame. One pass right computes the output; one pass left computes every gradient.
  • The chain rule multiplies along paths. A node's gradient is its downstream gradient times its local derivative — nothing more.
  • Learning = gradient step, repeated. Every weight moves against its gradient by the learning rate; do it thousands of times and the network learns.
  • The learning rate is delicate. Too big overshoots, too small crawls — visible right here in a 9-weight network.
