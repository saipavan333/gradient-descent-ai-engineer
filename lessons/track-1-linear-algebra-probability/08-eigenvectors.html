Eigenvectors & eigenvalues

Almost every vector gets knocked off its line by a transformation. A precious few don’t — they only stretch. Those special directions, and their stretch factors, reveal the hidden skeleton of a matrix, and they are the mathematical heart of PCA.

Track 1 · Lesson 8 Concept + derivation + code ~30 min Builds on: Lessons 5–7

💡 Concept: the directions a matrix doesn’t rotate

Apply a matrix \(A\) to a typical vector and two things change: its length and its direction (it gets rotated). But for special vectors, the direction survives — the matrix only stretches or shrinks them along the line they already sit on. Such a vector is an eigenvector, and the factor by which it stretches is its eigenvalue, written \(\lambda\) (the Greek letter “lambda”). The defining equation is as simple as it is consequential:

$$ A\mathbf{v} = \lambda\,\mathbf{v}, \qquad \mathbf{v}\neq\mathbf{0}. $$
(★)

Read it carefully: applying the whole matrix \(A\) to \(\mathbf{v}\) (the left side) gives the same result as just scaling \(\mathbf{v}\) by the single number \(\lambda\) (the right side). We require \(\mathbf{v}\neq\mathbf{0}\) because the zero vector trivially satisfies the equation for any \(\lambda\) and tells us nothing. The eigenvalue \(\lambda\) can be greater than 1 (stretch), between 0 and 1 (shrink), negative (flip and stretch), or zero (collapse onto the origin).

🧮 Finding them: the characteristic equation

How do we hunt down these special vectors? Start from (★) and move everything to one side. Using the identity matrix \(I\) (so that \(\lambda\mathbf{v} = \lambda I\mathbf{v}\), which lets us factor):

$$ A\mathbf{v} - \lambda\mathbf{v} = \mathbf{0} \quad\Longrightarrow\quad (A - \lambda I)\,\mathbf{v} = \mathbf{0}. $$

This says the matrix \((A-\lambda I)\) sends a nonzero vector \(\mathbf{v}\) to zero — it collapses that direction. But from Lesson 7 we know a matrix can crush a nonzero vector to \(\mathbf{0}\) only when it is singular, i.e. when its determinant vanishes. That gives the key that unlocks the eigenvalues, the characteristic equation:

$$ \det(A - \lambda I) = 0. $$
(‡)

This is one equation in the unknown \(\lambda\). Solve it for \(\lambda\), then for each \(\lambda\) solve \((A-\lambda I)\mathbf{v}=\mathbf{0}\) to get the eigenvector(s). Let’s do it on our running example \(A=\begin{bmatrix}2&1\\1&2\end{bmatrix}\).

🧮 Worked example, every step

Form \(A-\lambda I = \begin{bmatrix}2-\lambda & 1\\ 1 & 2-\lambda\end{bmatrix}\). Its determinant (Lesson 7) is

$$ (2-\lambda)(2-\lambda) - (1)(1) = \lambda^2 - 4\lambda + 3 = (\lambda-3)(\lambda-1). $$

Setting it to zero gives the eigenvalues \(\lambda_1 = 3\) and \(\lambda_2 = 1\). Now find each eigenvector:

For \(\lambda=3\): \((A-3I)\mathbf{v} = \begin{bmatrix}-1&1\\1&-1\end{bmatrix}\mathbf{v}=\mathbf{0}\) means \(-v_1+v_2=0\), i.e. \(v_1=v_2\). So \(\mathbf{v}_1 \propto (1,1)\).

$$ \text{Check: } \begin{bmatrix}2&1\\1&2\end{bmatrix}\begin{bmatrix}1\\1\end{bmatrix} = \begin{bmatrix}3\\3\end{bmatrix} = 3\begin{bmatrix}1\\1\end{bmatrix}. \checkmark $$

For \(\lambda=1\): \((A-I)\mathbf{v}=\begin{bmatrix}1&1\\1&1\end{bmatrix}\mathbf{v}=\mathbf{0}\) means \(v_1+v_2=0\), so \(\mathbf{v}_2 \propto (1,-1)\). (Eigenvectors are directions, so any nonzero multiple counts.)

For a symmetric matrix like this one, the eigenvectors come out perpendicular, and they’re exactly the axes along which the transformation acts as a pure stretch — turning the unit circle into an ellipse whose axes point along the eigenvectors and whose radii are the eigenvalues:

🎯 Why eigenvalues govern the long run

Eigenvectors make repeated application trivial. If \(A\mathbf{v}=\lambda\mathbf{v}\), then applying \(A\) twice gives \(A^2\mathbf{v}=A(\lambda\mathbf{v})=\lambda^2\mathbf{v}\), and in general:

$$ A^k\mathbf{v} = \lambda^k\,\mathbf{v}. $$

Now imagine any starting vector written as a combination of eigenvectors. Each component gets multiplied by its \(\lambda^k\), so after many steps the term with the largest \(|\lambda|\) dominates and the vector swings toward that eigenvector. This is why eigenvalues control the long-run behavior of any process built on repeated multiplication: the stability of dynamical systems, the steady state of a Markov chain, and Google’s original PageRank (the dominant eigenvector of the web’s link matrix) all hinge on it. It also gives the simplest algorithm for the top eigenvector — just multiply repeatedly (“power iteration”), which you’ll code below.

🤖 Why this is the math behind PCA

Here is the payoff that motivates the whole track. Given a cloud of data, its covariance matrix \(\Sigma\) (Lesson 11) encodes how the data spreads in each direction. Its eigenvectors are the directions of greatest variance — the natural axes of the data — and its eigenvalues are how much variance lies along each. Principal Component Analysis is exactly this: compute the eigenvectors of the covariance matrix, keep the few with the largest eigenvalues, and you’ve found the best low-dimensional basis (Lesson 4) to describe your data. Eigenvectors of the Hessian similarly describe the curvature of a loss landscape, telling you which directions are steep or flat — central to understanding optimization in Track 2. Eigen-thinking is everywhere once you can see it.

🛠️ From scratch: eigenvalues, verification, power iteration

code/track-1/eigen.py
import numpy as np

A = np.array([[2.0, 1.0],
              [1.0, 2.0]])

# NumPy finds eigenvalues and (unit) eigenvectors for us
vals, vecs = np.linalg.eig(A)
print("eigenvalues:", vals)            # [3. 1.]
print("eigenvectors (columns):\n", vecs)

# Verify the definition A v = lambda v for each pair
for i in range(len(vals)):
    v, lam = vecs[:, i], vals[i]
    print(np.allclose(A @ v, lam * v))  # True, True

# Power iteration: find the DOMINANT eigenvector by repeated multiplication
def power_iteration(A, steps=50):
    v = np.random.randn(A.shape[0])
    for _ in range(steps):
        v = A @ v
        v = v / np.linalg.norm(v)       # renormalize so it doesn't blow up
    lam = (v @ A @ v) / (v @ v)         # Rayleigh quotient -> the eigenvalue
    return lam, v

lam, v = power_iteration(A)
print("dominant lambda ~", round(lam, 4))   # ~3.0
print("dominant vector ~", np.round(v, 3))  # ~ (0.707, 0.707) i.e. (1,1)/sqrt2

# A tiny PCA: the top eigenvector of the covariance is the main axis of the data
X = np.random.randn(500, 2) @ np.array([[3.0, 1.0],[1.0, 1.0]])  # correlated cloud
Sigma = np.cov(X.T)                     # 2x2 covariance matrix
evals, evecs = np.linalg.eigh(Sigma)    # eigh: for symmetric matrices
top = evecs[:, np.argmax(evals)]
print("main axis of the data ~", np.round(top, 3))
⚠️ Two practical notes

Use np.linalg.eigh (not eig) for symmetric matrices like covariances — it’s faster and returns real, orthogonal eigenvectors. And remember eigenvectors are only defined up to sign and scale: NumPy returns unit-length ones, but \((0.707,0.707)\) and \((-0.707,-0.707)\) are the same eigen-direction.

🔑 Key results to remember

Lock these in

  1. Definition: \(A\mathbf{v}=\lambda\mathbf{v}\), \(\mathbf{v}\neq\mathbf{0}\). Eigenvectors keep their direction; \(\lambda\) is the stretch factor.
  2. Find eigenvalues by solving the characteristic equation \(\det(A-\lambda I)=0\); then solve \((A-\lambda I)\mathbf{v}=\mathbf{0}\) for each eigenvector.
  3. Symmetric matrices have real eigenvalues and orthogonal eigenvectors (circle → ellipse along those axes).
  4. Powers: \(A^k\mathbf{v}=\lambda^k\mathbf{v}\); the largest \(|\lambda|\) dominates long-run behavior (power iteration, PageRank, stability).
  5. PCA = eigenvectors of the covariance matrix = the data’s principal axes; eigenvalues = variance along each.

🧩 Quiz

🧩 Check your understanding
Answer from memory, then read why.

Q1. Which vector is an eigenvector of \(A=\begin{bmatrix}2&0\\0&5\end{bmatrix}\)?

  • \((1,1)\)
    \(A(1,1)=(2,5)\), which is not a scalar multiple of \((1,1)\) — its direction changed. Not an eigenvector.
  • \((1,2)\)
    \(A(1,2)=(2,10)=(2,10)\); is that \(\lambda(1,2)\)? That would need \((2,10)=(\,\lambda,2\lambda)\), impossible (\(\lambda=2\) and \(\lambda=5\) at once). No.
  • \((0,1)\)
    Correct. \(A(0,1)=(0,5)=5\,(0,1)\), a pure stretch by \(\lambda=5\). For a diagonal matrix, the axes \((1,0)\) and \((0,1)\) are always eigenvectors.
  • \((2,5)\)
    \(A(2,5)=(4,25)\), not a multiple of \((2,5)\). Being built from the diagonal entries doesn’t make it an eigenvector.

Q2. The eigenvalues of \(A\) are found by solving which equation?

  • \(\det(A-\lambda I)=0\)
    Correct. \((A-\lambda I)\) must crush a nonzero eigenvector to \(\mathbf{0}\), which requires it to be singular — i.e. zero determinant.
  • \(\det(A)=\lambda\)
    The determinant equals the product of all eigenvalues, not a single \(\lambda\). And this isn’t how you solve for them.
  • \(A\lambda = I\)
    This is meaningless dimensionally (\(\lambda\) is a scalar). The eigenvalue condition involves \(A-\lambda I\), not \(A\lambda\).
  • \(A\mathbf{v}=\mathbf{0}\)
    That finds vectors sent to zero by \(A\) itself (the null space), which are eigenvectors only for the special case \(\lambda=0\).

Q3. \(A\mathbf{v}=\lambda\mathbf{v}\) with \(\lambda=0.5\). What happens to \(\mathbf{v}\) after applying \(A\) ten times?

  • It grows to about \(5\mathbf{v}\).
    \(\lambda<1\) shrinks, it doesn’t grow. Growth needs \(|\lambda|>1\).
  • It stays exactly \(\mathbf{v}\).
    Only \(\lambda=1\) leaves a vector unchanged. Here each application halves it.
  • It flips sign each time.
    Sign flips require a negative \(\lambda\). Here \(\lambda=0.5>0\), so no flipping — just shrinking.
  • It shrinks to \(0.5^{10}\mathbf{v}\approx 0.001\mathbf{v}\).
    Correct. \(A^{10}\mathbf{v}=\lambda^{10}\mathbf{v}=0.5^{10}\mathbf{v}\), a tiny vector. Components with the smallest \(|\lambda|\) vanish fastest.

Q4. In PCA, what do the eigenvectors of the covariance matrix represent?

  • The data points with the largest values.
    Eigenvectors are directions in feature space, not individual data points.
  • The directions along which the data varies most (its principal axes).
    Correct. The top eigenvectors point along maximum variance; their eigenvalues say how much variance lies in each. Keeping the top few reduces dimension with minimal information loss.
  • The mean of the dataset.
    The mean is a single point (Lesson 2); eigenvectors describe the shape of the spread around that mean, not its location.
  • The number of features.
    The count of eigenvectors equals the number of features, but each eigenvector is a meaningful direction, not the count itself.

✍️ Practice

Problem 1 · compute

Find the eigenvalues of \(A=\begin{bmatrix}4&1\\2&3\end{bmatrix}\) using the characteristic equation.

Show solution

\(A-\lambda I=\begin{bmatrix}4-\lambda&1\\2&3-\lambda\end{bmatrix}\). Its determinant is

$$ (4-\lambda)(3-\lambda) - (1)(2) = \lambda^2 - 7\lambda + 12 - 2 = \lambda^2 - 7\lambda + 10 = (\lambda-5)(\lambda-2). $$

So \(\lambda_1=5\) and \(\lambda_2=2\). (Sanity check: the eigenvalues should sum to the “trace” \(4+3=7\) ✓, and multiply to \(\det A = 12-2=10\) ✓ — two quick checks worth knowing.)

Problem 2 · derivation

Show that if \(\mathbf{v}\) is an eigenvector of \(A\) with eigenvalue \(\lambda\), then \(\mathbf{v}\) is also an eigenvector of \(A^2\), and find its eigenvalue.

Show solution

Apply \(A\) twice and use \(A\mathbf{v}=\lambda\mathbf{v}\) at each step:

$$ A^2\mathbf{v} = A(A\mathbf{v}) = A(\lambda\mathbf{v}) = \lambda(A\mathbf{v}) = \lambda(\lambda\mathbf{v}) = \lambda^2\mathbf{v}. $$

So \(\mathbf{v}\) is an eigenvector of \(A^2\) with eigenvalue \(\lambda^2\). \(\blacksquare\) The eigenvectors are unchanged; only the eigenvalues get squared. By induction this gives \(A^k\mathbf{v}=\lambda^k\mathbf{v}\), the fact behind power iteration.

Problem 3 · code

Use NumPy to find the eigenvalues/eigenvectors of \(\begin{bmatrix}4&1\\2&3\end{bmatrix}\) and verify \(A\mathbf{v}=\lambda\mathbf{v}\) for the dominant pair.

Show solution 
import numpy as np
A = np.array([[4.0, 1.0],[2.0, 3.0]])
vals, vecs = np.linalg.eig(A)
print(np.sort(vals))                    # [2. 5.]
i = np.argmax(vals)                     # index of dominant eigenvalue (5)
v, lam = vecs[:, i], vals[i]
print(np.allclose(A @ v, lam * v))      # True

The eigenvalues match the hand calculation \(\{2,5\}\), and the verification confirms the dominant eigenvector satisfies the defining equation.

Problem 4 · reasoning

A transition matrix for a Markov chain has a dominant eigenvalue of exactly \(1\) and all others with \(|\lambda|<1\). What does repeatedly applying it to any starting distribution converge to, and why?

Show solution

Write the starting distribution as a combination of eigenvectors. After \(k\) steps each component is scaled by \(\lambda^k\). Every component with \(|\lambda|<1\) decays toward zero as \(k\to\infty\), while the component along the \(\lambda=1\) eigenvector is preserved (\(1^k=1\)). So the distribution converges to (a multiple of) that dominant eigenvector — the chain’s steady-state distribution. This is precisely the PageRank idea: the steady state of “random surfer” transitions is the \(\lambda=1\) eigenvector, and it ranks the pages. The gap between \(1\) and the next-largest \(|\lambda|\) sets how fast it converges.
